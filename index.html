How to use

  • Drag the grip beside a task name to move that task up or down the list.
  • Drag the center of a task bar to move the whole task.
  • Drag the left or right handle to resize the start or end date.
  • Use the color swatch on each row to customise the task color.
  • Set the full plan window and zoom the timeline in or out from the chart header.
